<div class="panel">
  <div class="panel-header">
    What this project is
  </div>

  This project documents the transition of a working system from Windows to
  Linux. The goal is not just the technical change, but to capture planning,
  testing, and decisions required to make the transition reproducible.

  Testing is being performed in a virtual environment first, with the explicit
  goal of recreating the validated setup later on physical hardware.
</div>

---

<div class="panel">
  <div class="panel-header">
    Why this is being done this way
  </div>

  <ul>
    <li>The existing system works but is nearing end-of-support</li>
    <li>Changes are being tested safely in a virtual environment</li>
    <li>Documentation is required for confident rebuilds</li>
    <li>The project is managed intentionally, not as an ad-hoc change</li>
  </ul>
</div>
